AI Spam Email Detection DevOps Project

About Project
This project detects whether a message is spam or not spam using machine learning. It also shows basic DevOps concepts using GitHub and CI CD.

Features
Detects spam messages
Uses real dataset
Simple web interface
Automatic process using GitHub Actions

Tools Used
Python
Scikit learn
Flask
Pandas
GitHub
GitHub Actions

How Project Works
1. Dataset is loaded
2. Text is converted into numbers
3. Model is trained
4. Model is saved
5. User enters message on website
6. Model predicts spam or not spam

DevOps Part
Code is uploaded on GitHub
When code is pushed pipeline runs automatically
Model training happens automatically

How to Run

Install libraries
pip install -r requirements.txt

Train model
python train_model.py

Run app
python app.py

Open in browser
http://127.0.0.1:5000/

Example

Input
Win money now

Output
Spam

Project Files
app.py web app

Conclusion
This project shows how AI model can be used with DevOps tools for automation and deployment
